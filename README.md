# K-Samplers — Virtual Instruments for Kadabra

K-Samplers are a series of virtual instruments designed with Kadabra players in mind.
Each instrument comes with special features and presets that bring Kadabra's motion
sensing to life — letting you shape sound expressively by spinning, tilting, and bowing
the instrument in real time.

While K-Samplers are built for Kadabra, they work as standard DAW plugins in any
compatible host. Supported formats include VST3 and others — see
[system requirements](https://www.innovativemusicalinstruments.com/ksamplers) for details.

---

## Built With

- [HISE](https://hise.dev/) — Hart Instruments Sampler Engine
- [JUCE](https://juce.com/) — Cross-platform audio application framework

## About IMI

K-Samplers are developed by [Innovative Musical Instruments (IMI)](https://www.innovativemusicalinstruments.com),
collaborators of Tribal Tools [Tribal Tools](https://www.tribal-tools.com),
